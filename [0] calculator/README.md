## 💡 Sobre o projeto

Primeiro projeto do **Dev Starter Pack**.

A missão aqui foi construir uma **calculadora funcional usando apenas HTML, CSS e JavaScript**.

Parece simples, mas é um ótimo exercício para entender como:

- o usuário interage com a interface
- o JavaScript processa essa interação
- a tela é atualizada com o resultado

<br>

## 🧠 Conceitos praticados

- estrutura de páginas HTML
- estilização com CSS
- seleção de elementos no DOM
- eventos de clique
- manipulação dinâmica da interface
- lógica de operações matemáticas
- controle de estado da aplicação

<br>

## 🔧 Tecnologias utilizadas

- HTML
- CSS
- JavaScript
